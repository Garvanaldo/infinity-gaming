**Infinity Gaming** is a serverless, cloud-synced management dashboard designed for gaming cafes. It allows the owner to control PC availability, track revenue, and manage bookings remotely from a smartphone while customers view a live "Kiosk" display at the shop.

The system uses **Google Firebase Realtime Database (Singapore Region)** to ensure that when a command is sent from the Admin Phone, the Shop PC updates instantly (latency < 100ms).
